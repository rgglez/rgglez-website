# Rodolfo González González

Allá por la segunda mitad de los años 1990 comencé a experimentar con las
entonces nuevas tecnologías para la WWW (HTML, JavaScript, CGI y Perl).
Construí mi sitio web y varios sitios más sobre mis aficiones. Pero, a
comienzos de los años 2000, perdí el interés y el tiempo para darles
continuidad. Ahora decidí regresar a ello.
