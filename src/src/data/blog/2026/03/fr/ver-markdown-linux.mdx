## Qu'est-ce que markdown ?

[Markdown](https://fr.wikipedia.org/wiki/Markdown) est un langage de balisage léger créé par John Gruber et Aaron Swartz qui vise à obtenir une lisibilité et une facilité de publication maximales aussi bien en entrée qu'en sortie, en s'inspirant de nombreuses conventions existantes pour baliser des messages électroniques en texte brut. John Gruber a créé le langage markdown en 2004, avec une aide importante d'Aaron Swartz pour la syntaxe. L'objectif de Gruber était que les gens « puissent écrire en utilisant un format de texte brut facile à lire, facile à écrire et avec la possibilité de convertir leur document en HTML valide ».

Les fichiers markdown peuvent être convertis en une représentation visuelle en HTML, mais aussi en PDF, par exemple.

## Comment afficher des fichiers markdown sous Linux

Supposons que nous ayons le document markdown suivant dans un fichier appelé `test.md` :

```markdown
# Document

*Bonjour le monde*

## Liste de courses

> Je dois acheter ce qui suit :

- Savon pour le chien.
- Nourriture pour le chat.
```

Étant un fichier texte, il peut être ouvert avec n'importe quel éditeur. Mais si nous souhaitons voir sa représentation graphique ou sa version finale publiable, il existe plusieurs façons de le faire sous Linux :

### pandoc

Depuis la ligne de commande, nous pouvons utiliser le programme [`pandoc`](https://pandoc.org/) pour convertir le fichier en PDF, puis utiliser n'importe quel visionneur pour ouvrir le fichier PDF résultant :

```bash
pandoc test.md -o test.pdf
okular test.pdf
```

### Okular

Si nous ne souhaitons pas convertir le document markdown en PDF, nous pouvons utiliser directement Okular avec le plugin approprié. Sur Debian et dérivés :

```bash
sudo apt install okular okular-extra-backends
okular test.md
```

Généralement, [`okular`](https://okular.kde.org/) est installé avec KDE, mais il faut installer séparément le paquet `okular-extra-backends`.

### Visual Studio Code

Il suffit d'ouvrir le fichier markdown dans l'éditeur et de cliquer sur le bouton : ![prévisualisation](@/assets/posts/2026/03/ver-markdown-linux/vscode.png) généralement situé dans le coin supérieur droit.
