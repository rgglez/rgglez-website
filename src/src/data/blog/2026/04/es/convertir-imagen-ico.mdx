## Introducción

Haciendo las imágenes de este, su blog de confianza, me vi en la necesidad de
convertir un PNG a ICO. Resulta que en Linux es muy fácil, usando
<a href="https://imagemagick.org/#gsc.tab=0" target="_blank">ImageMagick</a>.

## Pasos

Instalar imagemagick:

```bash
sudo apt install imagemagick
```

Convertir el PNG o JPEG (de 256x256px) a ICO:

```bash
convert input.png -define icon:auto-resize=256,64,48,32,16 favicon.ico
```

donde:

* `-define icon:auto-resize=256,64,48,32,16` guarda múltiples iconos de esos tamaños en el archivo `.ico`.

Y ya, es todo.
